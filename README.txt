COMMANDS

/filter <word-to-filter>
This commands hides the <word-to-filter>
Sending the command again will undo the filter
Alias: /delfilter

/replace <word-to-replace>;<word-to-show>
This commands hides <word-to-replace> 
and replaces it with <word-to-show> 
Sending the command again will undo the replace
Undoing can also be done by sending /replace <word-to-replace>
Alias: /delreplace
